# 🏭 IndustrialChain

Plataforma descentralizada para protección de propiedad industrial usando Solana.

---

## 🚀 Características

- Protección de IP mediante hash SHA-256
- Registro en blockchain (Solana)
- Almacenamiento seguro (Supabase)
- Chat cliente ↔ taller
- UI moderna tipo SaaS

---

## 🧱 Arquitectura

Frontend: Next.js  
Backend: Rust + Supabase 
Blockchain: Solana + Anchor
